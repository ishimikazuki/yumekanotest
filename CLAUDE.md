# Claude Code 設定

## 共通ルール
`PROJECT.md` の方針に従うこと。

## 作業記録
- 作業後は `WORKLOG.md` に追記する（日付と [Claude] タグをつける）
- 計画変更は `PLAN.md` に反映する

## Claudeの強みを活かす
- 不明点があれば積極的に質問する
- planモードで計画を立てたら `PLAN.md` に反映する
- 複雑な設計判断や探索的なリファクタで力を発揮する
- コードを読んで全体像を把握してから作業する

## planモードについて
計画を立てる際は `PLAN.md` を参照し、完了したら更新すること。
Codexとの連携のため、計画は共通フォーマットで記載する。
